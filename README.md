# SeaNest
SeaNest is a website-based system designed for the mapping and archiving of sea turtle nesting areas.  
The platform supports conservation efforts by providing structured, accessible, and visualized nesting data.

## Features
- Interactive mapping of nesting areas
- Archiving of nesting records and related media
- Researcher-friendly data presentation
- Conservation-focused design

## Purpose
This project was developed as part of a senior high school STEM research initiative, with the goal of supporting marine biodiversity conservation and data accessibility.

## Disclaimer
Exact nesting locations may be generalized or restricted to protect sensitive wildlife data.
